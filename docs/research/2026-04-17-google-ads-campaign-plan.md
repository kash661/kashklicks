# Google Ads Campaign Plan — AD Photography Toronto
*Prepared 2026-04-17 — Phase 1 local-wedding acquisition*

## Campaign objectives

- **Primary:** Drive inquiry form submissions from couples getting married in Toronto/GTA
- **Secondary:** Capture venue-specific intent (Casa Loma, city hall, Distillery, churches)
- **Exclude:** Overseas/destination wedding intent (already overserved by IG boost channel)

## Campaign settings

| Setting | Value |
|---|---|
| Campaign type | Search Network only |
| Network | Google Search (NO search partners, NO display) |
| Location targeting | Toronto + GTA (radius 40km from downtown Toronto) |
| Location option | "People in or regularly in" (not "interested in") — critical for local-only |
| Language | English |
| Bidding strategy | Manual CPC (max $3/click) for first 30 days → switch to Maximize Conversions once 15+ conversions tracked |
| Daily budget | $12/day ($350/mo ÷ 30) |
| Ad rotation | Optimize: prefer best performing ads |
| Ad schedule | 24/7 for first 30 days, then optimize after analysis |

## Landing page spec — `/intimate-wedding-toronto`

**This MUST be built before launching ads.** Sending ads to the current homepage cuts conversion ~50%.

### Structure (top to bottom)

1. **Hero section**
   - Serif H1: *"For Toronto couples who want the day to feel like theirs."*
   - Sans sub: *"Solo editorial wedding photography + film. Church, civil, backyard, small venue."*
   - Primary CTA button: `Book a Consultation`
   - Background: one striking Toronto-venue image (Casa Loma, local church interior, or City Hall arch)

2. **Approach (3-column grid)**
   - **One photographer.** No crew, no rush. One camera, one point of view.
   - **Photo + film.** The same solo photographer, same unified aesthetic, delivered together.
   - **Made here.** Local venues. Local families. Local stories.

3. **Recent work**
   - 4–6 thumbnails, ALL from Toronto/GTA weddings (not destination, not pre-wedding)
   - Each links to the couple's gallery
   - Caption each with the venue name

4. **Investment (pricing block)**
   - Recommended copy: *"Intimate wedding collections start at $3,500 CAD (4hr photo) and $5,500 CAD (photo + film). Each wedding is scoped to your day — consultation first, package second."*
   - Inquiry CTA below

5. **Testimonials (3 max)**
   - ONLY from couples who married in Toronto/GTA
   - Attribute with venue: *"Sarah & James — Allan Gardens"*

6. **FAQ (4 questions, accordion)**
   - Do you offer pre-wedding? (Yes, as part of wedding packages)
   - Do you travel? (GTA included. Canada-wide available)
   - What's included in photo + film? (Both same-day, same photographer, unified edit)
   - How soon should we book? (6–12 months typical)

7. **Final inquiry form**
   - Fields: Name, Email, Ceremony date (month/year), **Ceremony location** (free text + asterisk note "city/venue"), Optional story
   - Submit CTA: `Start the Conversation`

### Critical design rules
- No pre-wedding/engagement imagery in hero or above the fold. Wedding-day shots only.
- No "Indian wedding" or "South Asian" language anywhere.
- Venue names in alt text and image captions (SEO + local relevance signals).
- Mobile first — 60%+ of Google Ads traffic will be mobile.

## Conversion tracking (non-negotiable before launch)

Set up these events in Google Ads (via GA4 or direct gtag):

1. **Form submission** — primary conversion (value: $100 proxy, adjust later)
2. **Calendly booking** — primary conversion (value: $200 proxy)
3. **Phone call via extension** — secondary conversion (value: $50)
4. **Email click** — micro-conversion (value: $10)
5. **Scroll depth 75%** on landing page — micro-conversion (for optimization signal)

Without conversion tracking, Google Ads can't optimize → you waste spend. Don't skip this.

---

## Launch sequence (do in this order)

1. **Week 1:** Build `/intimate-wedding-toronto` landing page
2. **Week 1:** Set up GA4 + Google Ads conversion events
3. **Week 2:** Create Google Ads account (if not already), add campaign structure above
4. **Week 2:** Write/upload ad copy, keywords, negatives, extensions
5. **Week 2:** Launch campaign at $12/day, Manual CPC, 3 ad groups
6. **Weeks 3–4:** Daily monitoring. Kill keywords with >$15 spend and 0 conversions.
7. **Day 30:** Review. If ≥5 conversions tracked, switch bidding to Maximize Conversions. Add Ad Group 4 (elopement) if budget allows.
8. **Day 60:** Decide whether to expand to Ad Group 5 (broad "toronto wedding photographer") based on cost per conversion.

## Success thresholds (Month 1 targets)

| Metric | Target |
|---|---|
| Impressions | 3,000+ |
| Clicks | 150+ |
| CTR | ≥5% |
| Cost/click avg | ≤$2.50 |
| Conversions (form submissions + bookings) | 5+ |
| Cost/conversion | ≤$70 |
| Local-wedding inquiries (from form field) | 3+ |

If cost/conversion is above $70 after 30 days, the diagnosis is either (a) landing page conversion rate, (b) wrong keywords, or (c) ad copy not resonating. Each has different fixes — analyze in that order.

---

## Post-launch monitoring checklist

**Daily (first 2 weeks, 5 min):**
- Any keywords with >$10 spend + 0 conversions? Pause.
- New search terms in the report that aren't in the keyword list? Add as negative or expand.

**Weekly (ongoing, 20 min):**
- CTR by ad group — if <3%, revise ad copy
- Cost per click trending up? Add more negative keywords
- Landing page bounce rate — if >70%, the page isn't matching ad promise

**Monthly:**
- Full performance review
- Rebalance ad group budgets based on cost/conversion
- Decide if Phase 2 budget shift is warranted
